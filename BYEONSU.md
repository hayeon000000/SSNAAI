## 🔧 sensor_clean.csv

### 개요
- **수집 기간**: 2026-06-05 ~ 2026-06-14 (10일)
- **수집 주기**: 약 1분 1회
- **총 행 수**: 4,391행 (원본 16,575행 → 전처리 후)
- **수집 위치**: 성신여대 엘리베이터 앞

### 변수 설명

| 변수명 | 타입 | 설명 | 예시 |
|--------|------|------|------|
| `timestamp` | datetime | 측정 시각 (YYYY-MM-DD HH:MM) | `2026-06-05 09:15` |
| `pir1` | int (0/1) | PIR 센서 1번 감지 여부. 엘베에서 약 2m 지점. 1=감지, 0=미감지 | `1` |
| `pir2` | int (0/1) | PIR 센서 2번 감지 여부. PIR1에서 약 2m 지점(총 4m). 1=감지, 0=미감지 | `0` |
| `ultra1_dist` | float | 초음파 센서 1번 측정 거리 (cm). 엘베 마주보는 벽 설치. 사람 없을 때 평균 141cm | `98.50` |
| `ultra2_dist` | float | 초음파 센서 2번 측정 거리 (cm). 엘베 마주보는 벽 설치. 사람 없을 때 평균 153cm | `141.30` |
| `ultra1_detect` | str | 초음파 1번 감지 여부 문자열 | `detect` / `none` |
| `ultra2_detect` | str | 초음파 2번 감지 여부 문자열 | `detect` / `none` |
| `status` | str | 아두이노 원본 혼잡도 판별값 (영문) | `Not crowded` / `Normal` / `Crowded` |
| `hour` | int | 측정 시각의 시 (0~23) | `9` |
| `minute` | int | 측정 시각의 분 (0~59) | `15` |
| `day_of_week` | int | 요일 (0=월요일 ~ 6=일요일) | `0` |
| `day_name` | str | 요일 영문명 | `Monday` |
| `date` | date | 측정 날짜 | `2026-06-05` |
| `status_kor` | str | 재정의된 혼잡도 (한국어) | `여유` / `보통` / `혼잡` |

### 혼잡도 기준 (status_kor)

```
혼잡: PIR1=1 AND PIR2=1
      OR (ultra1_dist < 110cm AND ultra2_dist < 120cm)

보통: PIR1=1
      OR ultra1_dist < 110cm
      OR ultra2_dist < 120cm

여유: 위 조건 모두 해당 없음
```

> 초음파 기준값은 측정된 평균 거리(ultra1: 141cm, ultra2: 153cm)에서
> 30cm 이상 가까울 경우 사람 있음으로 판단

### 혼잡도 분포

| 혼잡도 | 건수 | 비율 |
|--------|------|------|
| 여유 | 3,439 | 78.3% |
| 보통 | 839 | 19.1% |
| 혼잡 | 113 | 2.6% |
