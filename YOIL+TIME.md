## 💡 주요 인사이트

### 1. 금요일 야간이 가장 혼잡
```
금요일 18:50 ~ 19:30 구간
혼잡 비율 55~80%로 압도적 1위
→ 야간 수업 시작 전 학생 집중
```

### 2. 목요일 오전 혼잡
```
목요일 10:20 ~ 11:00 구간
2교시 → 3교시 이동 시간대 혼잡
```

### 3. 주말도 혼잡 구간 존재
```
토요일 16:10 혼잡 60%
주말 수업 학생들의 엘베 이용 집중
```

### 4. 월~수 데이터 신뢰도 낮음
```
월~수 혼잡 0%는 실제 여유가 아닐 수 있음
센서 미작동 구간과 일치 → 추가 수집 필요
```
